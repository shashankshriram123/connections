# 📈 Social Graph Visualizer

A lightweight web app that lets you map and visualize how people are connected based on their **Instagram handles**. Users can enter their handle and the handles of friends, and see the network graph rendered in real-time.

---

## 🔧 Features

- Add yourself and your friends via Instagram handles
- Simple FastAPI backend with SQLite for persistent data storage
- Visualize social connections as a force-directed graph (D3.js)
- Profanity filtering using `better-profanity`
- (Optional fallback) Username syntax validation and profile link preview
